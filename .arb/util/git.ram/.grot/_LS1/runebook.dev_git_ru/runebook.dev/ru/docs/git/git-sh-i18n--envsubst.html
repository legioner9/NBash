git-sh-i18n--envsubst

Name

git-sh-i18n—​envsubst — собственный envsubst(1) Git для резервных вариантов i18n.

Synopsis

eval_gettext () {
        printf "%s" "$1" | (
                export PATH $(git sh-i18n--envsubst --variables "$1");
                git sh-i18n--envsubst "$1"
        )
}

Description

Это не та команда, которую хотел бы выполнить конечный пользователь. Всегда. Эта документация предназначена для людей, которые изучают сценарии сантехники и/или пишут новые.

git sh-i18n--envsubst — это урезанная копия программы GNU envsubst(1) , созданная в Git, которая поставляется с пакетом GNU gettext. Он используется внутри git-sh-i18n[1] для интерполяции переменных, передаваемых в функцию eval_gettext .

Никаких promises не сказано об интерфейсе или о том, что эта программа не исчезнет без предупреждения в следующей версии Git. Не используйте это.
