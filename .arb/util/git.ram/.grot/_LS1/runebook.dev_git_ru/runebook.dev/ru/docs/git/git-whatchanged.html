git-whatchanged

Name

git-whatchanged — Показать журналы с различиями, которые вносит каждый коммит.

Synopsis

git whatchanged <option>…​

Description

Показывает журналы коммитов и результаты различий, вносимые каждым коммитом.

Новым пользователям рекомендуется вместо этого использовать git-log[1] . Команда whatchanged по сути такая же, как и git-log[1] , но по умолчанию показывает необработанные выходные данные разницы формата и пропускает слияния.

Команда сохранена в первую очередь по историческим причинам; Пальцы многих людей, которые выучили Git задолго до изобретения git log , читая список рассылки ядра Linux, обучены его печатать.

Examples

git whatchanged -p v2.6.12.. include/scsi drivers/scsi

Показывать в виде исправлений коммиты, начиная с версии v2.6.12 , которые изменили любой файл в подкаталогах include/scsi или driver/scsi.

git whatchanged --since="2 weeks ago" -- gitk

Показать изменения за последние две недели в файле gitk . "--" необходим, чтобы не путать с веткой gitk .
