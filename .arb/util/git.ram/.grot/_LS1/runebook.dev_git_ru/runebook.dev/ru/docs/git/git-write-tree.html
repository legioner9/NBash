git-write-tree

Name

git-write-tree — Создать объект дерева из текущего индекса

Synopsis

git write-tree [--missing-ok] [--prefix=<prefix>/]

Description

Создает объект дерева, используя текущий индекс. Имя нового объекта дерева выводится в стандартный вывод.

Индекс должен находиться в полностью объединенном состоянии.

Концептуально git write-tree синхронизирует() содержимое текущего индекса в набор файлов дерева. Чтобы это соответствовало тому, что на самом деле находится в вашем каталоге прямо сейчас, вам необходимо выполнить фазу git update-index до того, как вы выполнили git write-tree .

Options

--missing-ok

Обычно git write-tree гарантирует, что объекты, на которые ссылается каталог, существуют в базе данных объектов. Эта опция отключает эту проверку.

--prefix=<prefix>/

Записывает объект дерева, представляющий подкаталог <prefix> . Это можно использовать для записи объекта дерева для подпроекта, который находится в указанном подкаталоге.
