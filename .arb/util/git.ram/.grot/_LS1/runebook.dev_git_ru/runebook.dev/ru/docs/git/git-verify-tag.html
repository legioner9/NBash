git-verify-tag

Name

git-verify-tag — проверить подпись тегов GPG.

Synopsis

git verify-tag [-v | --verbose] [--format=<format>] [--raw] <tag>…​

Description

Проверяет подпись gpg, созданную git tag .

Options

--raw

Распечатайте необработанный вывод состояния gpg в виде стандартной ошибки вместо обычного удобочитаемого вывода.

-v
--verbose

Распечатайте содержимое объекта тега перед его проверкой.

<tag>…

Идентификаторы SHA-1 объектов тегов Git .
