git-verify-pack

Name

git-verify-pack — проверить упакованные архивные файлы Git .

Synopsis

git verify-pack [-v | --verbose] [-s | --stat-only] [--] <pack>.idx…​

Description

Считывает указанный файл idx для упакованного архива Git , созданного с помощью команды git pack-objects , и проверяет файл idx и соответствующий файл пакета.

Options

<pack>.idx …

Файлы idx для проверки.

-v
--verbose

После проверки пакета покажите список объектов, содержащихся в пакете, и гистограмму длины дельта-цепи.

-s
--stat-only

Не проверяйте содержимое упаковки; показывать только гистограмму длины дельта-цепи. В --verbose также отображается список объектов.

--

Не интерпретируйте дополнительные аргументы как варианты.

Output format

При указании опции -v используется следующий формат:

SHA-1 type size size-in-packfile offset-in-packfile

для объектов, которые не разделены в пакете, и

SHA-1 type size size-in-packfile offset-in-packfile depth base-SHA-1

для объектов, которые являются дельтифицированными.
