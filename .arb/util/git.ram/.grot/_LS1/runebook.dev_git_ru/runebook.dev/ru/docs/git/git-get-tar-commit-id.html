git-get-tar-commit-id

Name

git-get-tar-commit-id — извлечь идентификатор фиксации из архива, созданного с помощью git-archive

Synopsis

git get-tar-commit-id

Description

Считайте tar-архив, созданный git archive , из стандартного ввода и извлеките хранящийся в нем идентификатор фиксации. Он считывает только первые 1024 байта входных данных, поэтому на время его выполнения размер tar-архива не сильно влияет.

Если идентификатор фиксации не найден, git get-tar-commit-id спокойно завершает работу с кодом возврата 1. Это может произойти, если архив не был создан с использованием git archive или если первым параметром git archive был идентификатор дерева вместо идентификатора фиксации или тега.
