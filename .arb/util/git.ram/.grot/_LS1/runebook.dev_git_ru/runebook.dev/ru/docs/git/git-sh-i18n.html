git-sh-i18n

Name

git-sh-i18n — код установки Git i18n для сценариев оболочки.

Synopsis

. "$(git --exec-path)/git-sh-i18n"

Description

Это не та команда, которую хотел бы выполнить конечный пользователь. Всегда. Эта документация предназначена для людей, которые изучают сценарии Porcelain и/или пишут новые.

Скриптлет 'git sh-i18n предназначен для использования (с использованием . ) фарфоровых программ Git, реализованных в сценарии оболочки. Он предоставляет оболочки для функций GNU gettext и eval_gettext , доступных через сценарий gettext.sh , и обеспечивает сквозные резервные варианты в системах без GNU gettext.

Functions

gettext

В настоящее время фиктивная функция падения реализована как оболочка вокруг printf(1) . В более поздней версии будет заменен настоящей реализацией gettext.

eval_gettext

В настоящее время фиктивная сквозная функция реализована как оболочка вокруг printf(1) с переменными, расширенными с помощью помощника git-sh-i18n--envsubst[1] . В более поздней версии будет заменен настоящей реализацией gettext.
