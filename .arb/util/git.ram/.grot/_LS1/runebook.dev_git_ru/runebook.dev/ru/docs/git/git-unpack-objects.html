git-unpack-objects

Name

git-unpack-objects — Распаковать объекты из запакованного архива

Synopsis

git unpack-objects [-n] [-q] [-r] [--strict]

Description

Считайте упакованный архив (.pack) со стандартного ввода, развернув содержащиеся в нем объекты и записав их в репозиторий в формате "loose" (один объект на файл).

Объекты, которые уже существуют в репозитории, не будут распакованы из пакфайла. Следовательно, ничего не будет распаковано, если вы используете эту команду для пак-файла, существующего в целевом репозитории.

См. git-repack[1] для получения информации о вариантах создания новых пакетов и замены существующих.

Options

-n

Пробный прогон. Проверьте файл пакета, не распаковывая объекты.

-q

Команда обычно показывает прогресс в процентах. Этот флаг подавляет это.

-r

При распаковке поврежденного паковочного файла команда прекращает работу при первом повреждении. Этот флаг говорит ему продолжать работу и приложить все усилия для восстановления как можно большего количества объектов.

--strict

Не пишите объекты с неработающим контентом или ссылками.

--max-input-size=<size>

Умрет, если упаковка больше <размера>.
