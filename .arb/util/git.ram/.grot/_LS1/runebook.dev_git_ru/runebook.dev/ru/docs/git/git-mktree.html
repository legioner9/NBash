git-mktree

Name

git-mktree — построить древовидный объект из текста в формате ls-tree.

Synopsis

git mktree [-z] [--missing] [--batch]

Description

Считывает стандартный ввод в нерекурсивном формате вывода ls-tree и создает объект дерева. Порядок записей дерева нормализуется mktree, поэтому предварительная сортировка входных данных не требуется. Имя построенного объекта дерева записывается в стандартный вывод.

Options

-z

Вместо этого прочитайте вывод ls-tree -z с нулевым завершением.

--missing

Разрешить отсутствующие объекты. Поведение по умолчанию (без этой опции) заключается в проверке того, что хэш каждой записи дерева идентифицирует существующий объект. Эта опция не влияет на обработку записей gitlink (также известных как "submodules" ), отсутствие которых всегда допускается.

--batch

Разрешить построение более одного древовидного объекта перед выходом. Каждое дерево отделяется одной пустой строкой. Последняя новая строка не является обязательной. Примечание. Если используется опция -z , строки завершаются NUL.
