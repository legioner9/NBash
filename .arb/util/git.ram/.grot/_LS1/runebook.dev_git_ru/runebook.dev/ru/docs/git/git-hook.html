git-hook

Name

git -крючок - Запуск крючков git

Synopsis

git hook run [--ignore-missing] [--to-stdin=<path>] <hook-name> [-- <hook-args>]

Description

Командный интерфейс для запуска перехватчиков git (см. githooks[5] ) для использования другими скриптовыми командами git .

Subcommands

run

Запустите перехватчик <hook-name> . См. githooks[5] для получения информации о поддерживаемых именах перехватчиков.

Любые позиционные аргументы перехватчика должны передаваться после обязательного -- (или --end-of-options , см. gitcli[7] ). См. githooks[5] для получения информации об аргументах, которые могут ожидать перехватчики (если таковые имеются).

Options

--to-stdin

Для "run" ; укажите файл, который будет передаваться в стандартный ввод перехватчика. Хук получит весь файл от начала до EOF.

--ignore-missing

Игнорируйте любой недостающий крючок, спокойно возвращая ноль. Используется для инструментов, которые хотят выполнить одноразовый проход крюка вслепую, который может присутствовать или отсутствовать.
