git-quiltimport

Name

git-quiltimport — применяет набор патчей quilt к текущей ветке.

Synopsis

git quiltimport [--dry-run | -n] [--author <author>] [--patches <dir>]
                [--series <file>] [--keep-non-patch]

Description

Применяет набор патчей quilt к текущей ветке Git , сохраняя границы патчей, порядок патчей и описания патчей, присутствующие в наборе quilt patchset.

Для каждого патча код пытается извлечь автора из описания патча. Если это не удается, он возвращается к автору, указанному с помощью --author. Если флаг --author не указан, отображается описание патча и пользователю предлагается интерактивно ввести автора патча.

Если тема не найдена в описании исправления, имя исправления сохраняется в виде темы в одну строку в описании Git .

Options

-n
--dry-run

Просмотрите патчи в серии и предупредите, если мы не сможем найти всю необходимую информацию для фиксации патча. На момент написания статьи предупреждается только об отсутствии информации об авторе.

--author Имя автора <Электронная почта автора>

Имя автора и адрес электронной почты, которые следует использовать, если в описании патча нет информации об авторе.

--patches <dir>

Каталог для поиска патчей quilt.

По умолчанию для каталога исправлений используется patches или значение переменной среды $QUILT_PATCHES .

--series <file>

Файл серии лоскутных одеял.

По умолчанию для файла серии используется <patches>/series или значение переменной среды $QUILT_SERIES .

--keep-non-patch

Передайте флаг -b в git mailinfo (см. git-mailinfo[1] ).
