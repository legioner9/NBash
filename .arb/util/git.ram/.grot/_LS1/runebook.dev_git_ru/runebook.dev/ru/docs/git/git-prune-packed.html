git-prune-packed

Name

git-prune-packed — удалить лишние объекты, которые уже есть в файлах пакета.

Synopsis

git prune-packed [-n | --dry-run] [-q | --quiet]

Description

Эта программа ищет в $GIT_OBJECT_DIRECTORY все объекты, которые в данный момент существуют в пакетном файле, а также в независимых каталогах объектов.

Все такие лишние объекты удаляются.

Пакет — это набор объектов, индивидуально сжатых с применением дельта-сжатия, хранящихся в одном файле со связанным индексным файлом.

Пакеты используются для снижения нагрузки на зеркальные системы, механизмы резервного копирования, дисковые хранилища и т. д.

Options

-n
--dry-run

На самом деле не удаляйте никакие объекты, а показывайте только те, которые могли бы быть удалены.

-q
--quiet

Подавить индикатор прогресса.
