git-merge-one-file

Name

git-merge-one-file — стандартная вспомогательная программа для использования с git-merge-index.

Synopsis

git merge-one-file

Description

Это стандартная вспомогательная программа, которую можно использовать с git merge-index для разрешения слияния после тривиального слияния, выполненного с помощью git read-tree -m .
