git-instaweb

Name

git-instaweb — мгновенно просматривать рабочий репозиторий в gitweb.

Synopsis

git instaweb [--local] [--httpd=<httpd>] [--port=<port>]
               [--browser=<browser>]
git instaweb [--start] [--stop] [--restart]

Description

Простой скрипт для настройки gitweb и веб-сервера для просмотра локального репозитория.

Options

-l
--local

Привязывайте веб-сервер только к локальному IP (127.0.0.1).

-d
--httpd

Командная строка демона HTTP, которая будет выполнена. Здесь можно указать параметры командной строки, а файл конфигурации будет добавлен в конец командной строки. В настоящее время поддерживаются apache2, Lighttpd, mongoose, plackup, python и webrick. (По умолчанию: светтпд)

-m
--module-path

Путь к модулю (нужен, только если httpd — Apache). (По умолчанию: /usr/lib/apache2/modules)

-p
--port

Номер порта, к которому будет привязан httpd. (По умолчанию: 1234)

-b
--browser

Веб-браузер, который следует использовать для просмотра страницы gitweb. Это будет передано вспомогательному сценарию git web--browse вместе с URL-адресом экземпляра gitweb. Дополнительную информацию об этом см. в git-web--browse[1] . Если сценарий завершится неудачно, URL-адрес будет выведен на стандартный вывод.

start
--start

Запустите экземпляр httpd и выйдите. При необходимости заново создайте файлы конфигурации для создания нового экземпляра.

stop
--stop

Остановите экземпляр httpd и выйдите. При этом не создаются файлы конфигурации для создания нового экземпляра и не закрывается браузер.

restart
--restart

Перезапустите экземпляр httpd и выйдите. При необходимости заново создайте файлы конфигурации для создания нового экземпляра.

Configuration

Вы можете указать конфигурацию в вашем .git/config.

[instaweb]
        local = true
        httpd = apache2 -f
        port = 4321
        browser = konqueror
        modulePath = /usr/lib/apache2/modules

Если переменная конфигурации instaweb.browser не установлена, вместо нее будет использоваться web.browser , если она определена. См. git-web--browse[1] для получения дополнительной информации об этом.
