git-citool

Name

git-citool — графическая альтернатива git-commit

Synopsis

git citool

Description

Графический интерфейс на основе Tcl/Tk для просмотра измененных файлов, помещения их в индекс, ввода сообщения о фиксации и записи новой фиксации в текущую ветку. Этот интерфейс является альтернативой менее интерактивной программе git commit .

git citool на самом деле является стандартным псевдонимом git gui citool . Дополнительную информацию см. в git-gui[1] .
