git-mailsplit

Name

git-mailsplit — простая программа разделения mbox для UNIX

Synopsis

git mailsplit [-b] [-f<nn>] [-d<prec>] [--keep-cr] [--mboxrd]
                -o<directory> [--] [(<mbox>|<Maildir>)…​]

Description

Разбивает файл mbox или Maildir на список файлов: "0001" "0002" .. в указанном каталоге, чтобы вы могли обрабатывать их дальше.

Important
 Разделение Maildir основано на сортировке имен файлов для вывода патчей в правильном порядке.

Options

<mbox>

Mbox-файл, который нужно разделить. Если не задано, mbox считывается со стандартного ввода.

<Maildir>

Корень Maildir для разделения. Этот каталог должен содержать подкаталоги cur, tmp и new.

-o<directory>

Каталог, в котором можно разместить отдельные сообщения.

-b

Если какой-либо файл не начинается со строки «От», предположим, что это одно почтовое сообщение, а не сигнализирует об ошибке.

-d<prec>

Вместо стандартных 4 цифр с ведущими нулями для сгенерированных имен файлов можно указать различную точность.

-f<nn>

Пропустите первые цифры <nn>, например, если указан -f3, нумерацию начните с 0004.

--keep-cr

Не удаляйте \r из строк, заканчивающихся на \r\n .

--mboxrd

Ввод имеет формат "mboxrd" , а экранирование строк "^>+From " инвертировано.
