git-count-objects

Name

git-count-objects — подсчитать количество распакованных объектов и их использование на диске.

Synopsis

git count-objects [-v] [-H | --human-readable]

Description

Подсчитывает количество распакованных объектных файлов и занятое ими дисковое пространство, чтобы помочь вам решить, когда лучше перепаковать их.

Options

-v
--verbose

Предоставляем более подробные отчеты:

подсчет: количество незакрепленных предметов

размер: дисковое пространство, занимаемое незакрепленными объектами, в КиБ (если не указан -H)

в упаковке: количество объектов в упаковке.

size-pack: дисковое пространство, занимаемое пакетами, в КиБ (если не указан -H)

в упаковке с черносливом: количество незакрепленных предметов, которые также присутствуют в упаковках. Эти объекты можно удалить с помощью git prune-packed .

мусор: количество файлов в базе данных объектов, которые не являются ни допустимыми свободными объектами, ни допустимыми пакетами.

size-garbage: дисковое пространство, занимаемое мусорными файлами, в КиБ (если не указан -H)

альтернативный: абсолютный путь к альтернативным объектным базам данных; может появляться несколько раз, по одной строке на каждый путь. Обратите внимание: если путь содержит непечатаемые символы, он может быть заключен в двойные кавычки и содержать escape-последовательности с обратной косой чертой в стиле C.

-H
--human-readable

Размеры печати в удобочитаемом формате
