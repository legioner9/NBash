git-check-mailmap

Name

git-check-mailmap — Показать канонические имена и адреса электронной почты контактов.

Synopsis

git check-mailmap [<options>] <contact>…​

Description

Для каждого «Имя <user@host>» или «<user@host>» из командной строки или стандартного ввода (при использовании --stdin ) найдите каноническое имя и адрес электронной почты человека (см. "Mapping Authors" ниже). Если они найдены, распечатайте их; в противном случае распечатайте ввод как есть.

Options

--stdin

Считайте контакты, по одному на строку, из стандартного ввода после исчерпания контактов, указанных в командной строке.

Output

Для каждого контакта выводится одна строка, завершающаяся новой строкой. Если имя предоставлено или известно mailmap , печатается «Имя <user@host>»; в противном случае печатается только «<user@host>».

Configuration

См. mailmap.file и mailmap.blob в git-config[1] , чтобы узнать, как указать пользовательский целевой файл или объект .mailmap .

Mapping authors

See gitmailmap[5].
