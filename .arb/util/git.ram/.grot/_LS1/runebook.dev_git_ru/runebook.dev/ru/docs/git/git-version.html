git-version

Name

git-version — вывести информацию о версии Git .

Synopsis

git version [--build-options]

Description

Если параметры не указаны, версия git выводится на стандартный вывод.

Обратите внимание, что git --version идентичен git version , поскольку первый внутренне преобразуется во второй.

Options

--build-options

Включите дополнительную информацию о том, как git был создан для диагностических целей.
