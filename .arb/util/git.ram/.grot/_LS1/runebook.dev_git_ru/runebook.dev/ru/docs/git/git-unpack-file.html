git-unpack-file

Name

git-unpack-file — создает временный файл с содержимым объекта.

Synopsis

git unpack-file <blob>

Description

Создает файл, содержащий содержимое большого двоичного объекта, указанного sha1. Он возвращает имя временного файла в следующем формате: .merge_file_XXXXX.

Options

<blob>

Должен быть идентификатор большого двоичного объекта
