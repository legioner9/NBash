git-update-server-info

Name

git-update-server-info — обновить файл вспомогательной информации, чтобы помочь тупым серверам.

Synopsis

git update-server-info [-f | --force]

Description

Простой сервер, который не генерирует пакеты «на лету», должен иметь некоторые вспомогательные информационные файлы в каталогах $GIT_DIR/info и $GIT_OBJECT_DIRECTORY/info, чтобы помочь клиентам узнать, какие ссылки и пакеты есть на сервере. Эта команда генерирует такие вспомогательные файлы.

Options

-f
--force

Обновите информационные файлы с нуля.

Output

В настоящее время команда обновляет следующие файлы. Пожалуйста, смотрите gitrepository-layout[5] для описания того, для чего они предназначены:

  • objects/info/packs

  • info/refs
